# lens

A multiformat internationalization (i18n) shard for Crystal. Supports GNU Gettext, JSON, YAML and much more!

## Installation

1. Add the dependency to your `shard.yml`:

   ```yaml
   dependencies:
     lens:
       github: syeopite/lens
   ```

2. Run `shards install`
